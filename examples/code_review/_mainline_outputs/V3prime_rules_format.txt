c2_counter_race_BEGIN
Issue (line 6): `self.count += 1` is not atomic â€” it is a read-modify-write that can interleave under threading, losing increments. The GIL serializes individual bytecodes but not the compound `+=`.
Fix: Guard with a lock:
```
def __init__(self):
    self.count = 0
    self._lock = threading.Lock()

def increment(self):
    with self._lock:
        self.count += 1
```
